## 🚀 Quick Start

### インストール
```bash
# フロントエンド
cd frontend
npm install

# バックエンド
cd backend
npm install
```

### 実行
```bash
# フロントエンド
npm start

# バックエンド
npm run dev
```

## 📖 User Guide

### 基本機能
1. **ワークアウト記録**: 日々の運動を簡単に記録
2. **統計表示**: 週次・月次の運動統計を確認
3. **Strava連携**: Stravaアカウントと自動同期
4. **目標設定**: 月次目標の設定と追跡
